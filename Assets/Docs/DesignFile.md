## New Card Game Try Project

### 游戏规则

当前卡牌游戏的基本游戏规则为：两人对战，轮流将手牌中的对象置于棋盘中，在双方放置完毕后对比两方棋子的战力大小（可以类比巫师3中的昆特牌）。

#### 棋盘

棋盘大小为4 * 8：即总共为4列8行，其中分为两个4 * 4的半区，对战的双方各自拥有一个半区。每一轮，出牌方都只能将自己的牌当作棋一个对象下在自己的半区棋盘内。最终进行战力大小比较的时候同样根据半区进行战力计算：即使当前本方半区内存在敌方棋子，其战力同样被计算在本方的总战力中。棋盘大致如下：

|行\列|1列|2列|3列|4列|
|---|---|---|---|---|
|8行|敌|敌|敌|敌|
|7行|敌|敌|敌|敌|
|6行|敌|敌|敌|敌|
|5行|敌|敌|敌|敌|
|分界|分界|分界|分界|分界|
|4行|我|我|我|我|
|3行|我|我|我|我|
|2行|我|我|我|我|
|1行|我|我|我|我|

#### 卡组准备

本卡牌游戏拥有不同阵营的卡组，每次玩家只能选择同一阵营的卡组进入战斗。玩家在游戏开始时即会获得一个阵营的所有基础牌（每一个阵营的基础牌数目均为20张）、后续的稀有卡牌及其他阵营的卡牌需要经过游戏中的搜索获得。

无论玩家拥有多少阵营卡牌，都只能选择最少16张、最多30张卡牌进入战斗，只有被选择的牌有可能在战斗中出现。

#### 打牌流程

在战斗开始时，系统会从选择加入战斗的牌中自动洗出10张加入你的手牌。双方根据抛硬币（随机）选择先出牌的一方，每次出牌方可以选择两种操作：

第一种操作为从手牌中选择一张牌，并选择棋盘中的一个没有其他单位的我方格子放入（有可能有牌能够直接放入敌方格子，但是绝大多数都只能放在我方格子中）。

第二种操作为选择结束回合，如果觉得本回合自己打出的牌已经足够并不想进行后续出牌，就可以选择结束回合，此后在本轮中只有敌方可以出牌，直到地方也选择结束回合。

每场战斗为三轮两胜，每一轮回合结束并不会补充手牌，只有特定手牌技能可能在轮间、轮中让玩家能够随机从未被开始时选择的牌加入手牌。
